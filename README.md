## Calculadora Web Simples

Esta é uma aplicação web simples de calculadora desenvolvida em:
* JavaScript 
* HTML 
* CSS.

### Funcionalidades

- Adição
- Subtração
- Multiplicação
- Divisão
- Limpeza de Tela (Clear Display)

### Pré-requisitos

- Navegador da web Google Chrome

### Como usar

1. Clone este repositório:

   ```bash
   git clone https://github.com/KaDelanegri/calculator-javascript.git
   ```

2. Navegue até o diretório da aplicação:

   ```bash
   cd calculator-javascript
   ```

3. Abra o arquivo `index.html` em seu navegador da web.

### Estrutura do Projeto

- `index.html`: Página principal da aplicação.
- `style.css`: Estilos CSS para a interface da calculadora.
- `script.js`: Lógica em JavaScript para realizar operações de cálculo.

### Design

O design da calculadora é minimalista e fácil de usar, com botões para cada operação e uma tela de exibição para mostrar o resultado.

### Desenvolvimento

O projeto foi desenvolvido utilizando JavaScript para a lógica das operações, HTML para a estrutura da página e CSS para o estilo da interface.
